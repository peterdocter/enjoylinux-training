# enjoylinux-training

国嵌核心技术团队在综合本身10余年开发经验和5年教学经验的基础上，制定出一套非常系统、完善的嵌入式Linux培训课程体系，该体系由

1.《嵌入式体验入门班》

2.《嵌入式Linux应用开发班》

3.《ARM系统进阶班》

4.《ARM系统深入班》

5.《嵌入式Linux内核驱动进阶班》

6.《嵌入式Linux内核驱动深入班》

7.《嵌入式Linux系统移植专题班》

8.《嵌入式Linux高级项目培训班》

8大课程组成，由简入深，循序渐进，通过系统的讲解，丰富的实验，真实的项目，让你从一位没有任何嵌入式基础的门外汉，一步步成长为一名合格的嵌入式Linux工程师。
